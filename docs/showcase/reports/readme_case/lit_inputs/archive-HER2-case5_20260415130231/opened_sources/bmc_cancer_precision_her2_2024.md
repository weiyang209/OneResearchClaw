# Precision HER2: a comprehensive AI system for accurate and consistent evaluation of HER2 expression in invasive breast cancer

**Source:** BMC Cancer, 2024
**URL:** https://link.springer.com/article/10.1186/s12885-024-12980-6
**Access date:** 2026-04-15

## Abstract

Precision HER2 is a comprehensive AI system for automated HER2 scoring on IHC whole-slide images. The system achieved high concordance with expert pathologists and improved scoring consistency across slides and laboratories. The system follows ASCO/CAP 2018/2023 guidelines and focuses on invasive tumour regions.

## Methodology

1. Whole-slide image analysis pipeline
2. Invasive tumour region detection
3. Individual tumour cell membrane staining classification
4. ASCO/CAP guideline-compliant scoring (0, 1+, 2+, 3+)
5. Multi-phase approach: filtering DCIS, analyzing positive controls

## Key Claims

- High concordance with expert pathologists
- Improved scoring consistency across slides and laboratories
- AI systems reduced misdiagnosis of HER2 1+ patients as HER2 0
- Enables more patients to benefit from antibody-drug conjugate (ADC) therapy

## Relevance

This paper represents the state-of-the-art in fully automated HER2 IHC scoring on whole-slide images, directly relevant to the clinical deployment context of the HER2 prediction framework in the source documents.
